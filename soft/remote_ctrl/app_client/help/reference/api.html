Приложение app_client при запуске автоматически исполняет файл client.lua. Все остальные файлы, если таковые будут, могут быть запущены оттуда. Исполняемые процедуры не обязательно должны выполнять возврат. Процедуры выполняются стеком. Другими словами, если при приходе команды выполняется какая-то процедура, то она приостанавливается, выполняется пришедшая команда, а потом исполнение прерванной процедуры возобновляется. Количество таких уровней вложенности может быть произвольным в разумных пределах.


app_client при соединении считывает из папки файл client.ini и получает из него параметры соединения.


jid_self

Этот параметр — логин пользователя на сервере XMPP.

password

Пароль пользователя на сервере XMPP.


host

В случае, если доменное имя сервера не может быть разрешено из логина, оно может быть явно указано параметром host.


tls

Опциональное задействование шифрования TLS при соединении. Для GoogleTalk должно быть равно true, иначе не подсоединится.


jid_dest

Логин управляемого приложения, которому будут отправляться команды.


update_dest

Параметр определяет, если пришедшая команда не от пользвателя jid_dest, то значение jid_dest обновляется.


На данный момент оборудование находится в разработке. Для отработки каких-либо алгоритмов вставлены эмуляции элементов управления на пульте робота — 4 кружка, которые временно заменяют настоящие джойстики.


Функциональность Lua расширена приведёнными ниже функциями


msleep( ms )

Остановка исполнения скрипта на указанное время в милисекундах.


сonnect()

Соединение с сервером XMPP / GoogleTalk.


isConnected()

Проверка наличия соединения. Возвращает true/false.


send( cmd )

Отправка команды


sendFile( fileName, destFileName )

Отправка файла. Первый аргумент — исходное имя отправляемого файла. Второй аргумент — имя файла при полкчении.


stop()

Остановка исполнения всех запущенных команд.


print( ... )

Вывод/печать аргументов.


joy( index )

Опрос состояний эмуляции джойстиков (кружки на экране). Аргумент — номер джойстика (от 1 до 4). Возвращает два числа — отклонение от центра в процентах.
